# IsThisAI Reddit Bot

## Setup

1. Create and activate a virtualenv.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Copy env template and fill credentials:
   ```bash
   cp .env.example .env
   ```
4. Run:
   ```bash
   python bot.py
   ```

## Environment variables

See `.env.example`.
